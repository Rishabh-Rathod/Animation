# Animation Demo

This repo contains a fresh React Native project with preinstalled Reanimated V2


After Cloning Repo 

1. Install dependencies:
```bash
> cd Animation && yarn
```

2. Run in Android
```bash
> yarn run android
```

If you want to install app on the iOS simulator:
```bash
npx pod-install
```
3. Run in IOS

```bash
> yarn run ios
```
